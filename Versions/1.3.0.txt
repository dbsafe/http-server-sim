Version 1.3.0

Added
- Added support for delaying responses before they are sent.
- Added `--DefaultDelayMin` and `--DefaultDelayMax` to configure delays for the default response.
- Added support for configuring delays per rule through the `delay` section in rule files.
- Added Swagger to the control endpoint.
- Added `GetAllRules`, `GetRule`, `RuleExists`, `DeleteRule`, and `UpdateRule` to the control endpoint and `HttpSimClient`.
